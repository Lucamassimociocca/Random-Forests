# Random-Forests (scikit-learn — Classification)
This repository includes a Random Forest classifier implemented via scikit-learn, serving as an ensemble alternative to the neural network for binary classification tasks. A Random Forest builds a large number of decision trees on bootstrap samples of the training data and aggregates their predictions by majority vote, making it naturally robust to overfitting and capable of capturing non-linear feature interactions without requiring input scaling or explicit interaction terms. The model is trained with 100 estimators and class_weight set to "balanced" to handle imbalanced targets; additionally, SMOTE is applied to the training data prior to fitting for explicit resampling. Prediction probabilities are extracted via predict_proba, enabling both hard classification and ranked scoring. Model performance is evaluated using a classification report, ROC AUC score, and a Lift & Cumulative Gains analysis — the latter sorts observations by predicted probability into deciles and measures how much better the model identifies true positives compared to random selection, which is particularly useful for translating model output into actionable ranked lists. Feature importance is available intrinsically through mean decrease in impurity (Gini importance) across all trees. As with the neural network scripts, the target variable, feature list, and any interaction terms are defined in a clearly marked section at the top of the notebook and can be replaced for any binary classification problem on tabular data.
